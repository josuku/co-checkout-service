# co-checkout-service

## Description

Microservice implemented in NestJS.
Communicates with NestJS microservices using the request-response message style. 
Simulates a checkout process.
Exposes a public API in port 3001.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
